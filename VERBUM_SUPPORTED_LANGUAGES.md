## Usage Notes

- Use 2-letter codes (e.g., `en`, `es`, `pt`) for Verbum API calls
- For Chinese variants, use `zh-Hans` (Simplified) or `zh-Hant` (Traditional)
- For French Canadian, use `fr-ca` instead of `fr`
- For Portuguese, use `pt` (Brazil) or `pt-pt` (Portugal)
- Total: 133 supported translation languages
